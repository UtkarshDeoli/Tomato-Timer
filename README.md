Getting Started
How to setup ?
Clone the repo
git clone https://github.com/UtkarshDeoli/zero_chat.git

cd to zero_chat cd zero_chat

Get Dependency flutter pub get

Create flutter create .

Run flutter run

Or just copy this to your terminal:
  git clone https://github.com/UtkarshDeoli/zero_chat.git
  cd zero_chat
  flutter pub get
  flutter create .
  flutter run
